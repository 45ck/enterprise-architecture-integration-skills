# Sample Flow

## Scenario
A university-scale platform needs to integrate a student portal, learning platform, identity provider, finance system, and notification services while preparing for more automated releases and a likely cloud migration.

## Suggested skill sequence
1. `enterprise-context-mapper`
2. `application-landscape-mapper`
3. `enterprise-domain-alignment-checker`
4. `rpc-vs-message-pattern-selector`
5. `service-contract-designer`
6. `orchestration-vs-choreography-reviewer`
7. `environment-topology-planner`
8. `ci-cd-readiness-checker`
9. `public-private-hybrid-cloud-selector`

## Expected artifacts
- enterprise context map
- application landscape and ownership map
- domain alignment review
- integration pattern decision
- service contracts for major interfaces
- workflow coordination recommendation
- environment topology plan
- CI/CD readiness report
- cloud hosting recommendation
